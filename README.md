# Cómo ejecutar la aplicación

1. Clonar el repositorio:

```bash
git clone <URL_DEL_REPOSITORIO>
cd twitch-analytics
```

2. Crear un archivo .env siguiendo el ejemplo de .env.example:

```bash
cp .env.example .env
```

3. Instalar dependencias y ejecutar la aplicación:

```bash
npm i 
node server.js
```
